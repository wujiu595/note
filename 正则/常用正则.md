## 密码

### go方案1 

假设密码验证做如下规则定义：

- 最短6位，最长16位 {6,16}
- 可以包含小写大母 [a-z] 和大写字母 [A-Z]
- 可以包含数字 [0-9]
- 可以包含下划线 [ _ ] 和减号 [ - ]

根据以上规则，很容易给出正则字面量定义如下：

```go
var pattern = `^[\w_-]{6,16}$`;
```

### 方案2

假设密码验证做如下规则定义：

- 最短6位，最长16位 {6,16}
- 必须包含1个数字
- 必须包含2个小写字母
- 必须包含2个大写字母
- 必须包含1个特殊字符

根据以上规则，很容易给出正则字面量定义如下：

```go
var pattern = `^[a-z0-9A-Z]+[- | a-z0-9A-Z . _]+@([a-z0-9A-Z]+(-[a-z0-9A-Z]+)?\\.)+[a-z]{2,}$`;
```

拆分查看

```go
(?=.{6,16}) 6~16位
(?=.*\d)  必须包含1个数字
(?=.*[A-Z]{2,}) 必须包含2个大写字母
(?=.*[a-z]{2,}) 必须包含2个小写字母
(?=.*[!@#$%^&*?\(\)]) 必须包含1个特殊字符
```
