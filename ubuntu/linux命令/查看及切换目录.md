## 查看文件路径

- 查看文件夹内容

```shell
ls [-a]
-a查看所有目录包括隐藏目录
```

- 使用文件定位符

把查看到内容写到一个文件种

```shell
ls > b.txt
```

- 显示文件详细信息

```shell
ls -all
```

- 查看文件夹的树型结构

```shell
tree
```

- 查看当前所在的路径

```
pwd
```

## 切换目录

- 返回上级目录

```shell
cd ..
```

- 返回到home目录

```shell
cd ~
```

- 返回上一次访问的目录

```shell
cd -
```

- 绝对路径

```shell
cd /`路径`
cd /etc
```

- 相对路径

```shell
cd `路径`
如:cd user
```
