### JavaScipt嵌入页面方式



#### 1. 行间事件(主要用于事件)

```javascript
<input type="button" name="" onclick="alert('ok!')">
```

#### 2. 页面script标签嵌入

```javascript
<script text="text/javascript">
    alert("ok!")
</script>    
```

#### 3.外部引入

```javascript
<script type="text/javascript" src="js/index.js"></script>
```
